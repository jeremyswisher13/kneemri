# UCLA Sports MRI Courses Medical QA

Generated: 2026-07-10

This folder is generated by `npm run qa:medical`.

## Files

- `packets/*.md`: human-readable course review packets.
- `packets/*.json`: machine-readable course packets for downstream tools.
- `review-items.csv`: spreadsheet-ready review queue with blank reviewer/source-note columns.
- `summary.json`: counts and automated diagnostics.
- `public/medical-qa/source-check-items.json`: admin-page source-check queue.

## Intended Workflow

1. Start with `summary.json` and resolve any diagnostic errors.
2. Sort `review-items.csv` by `risk` and `requires_source_check`.
3. Physician/MSK radiology reviewer fills `review_status`, `reviewer`, `reviewed_at`, `source_notes`, and `reviewer_notes`.
4. Apply content fixes in `src/content/**`.
5. Regenerate packets and confirm changed rows are reviewed.

## Important Limits

Automated QA can catch answer-key, attribution, and consistency issues, and it can flag high-risk/source-needed claims. It cannot prove that medical teaching content is correct. Final approval requires human clinical review with source notes.
